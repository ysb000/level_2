CSS 시작

작성순서는 항상 html->css 순이다.

css 작성 시 선택자부터 먼저 생각해야한다.

선택자 {속성:값;}

값 뒤엔 항상 ;이 붙어야한다.

속성 1개 작성할 때
선택자 {속성:값;}
속성 2개 작성할 때
선택자 {속성:값; 속성:값;}
  • CSS파일은
  • style 폴더에 저장해야한다.

  2. html을 선택자로 해서 css 작성하기
